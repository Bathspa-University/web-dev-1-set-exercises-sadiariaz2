HTML Tags

HTML tags that defines the structure of your web page.

  • <h1>: This defines the heading1
  • <div>: Defines a division or section
  • <p>: Defines a paragraph
  • <img>: Embeds an image
  • <a>: Defines a hyperlink
  • <ul>: Defines an unordered list
  • <ol>: Defines an ordered list
  • <table>: Defines a table
  • <form>: Defines a form
  • <input>: Defines an input field
  • <button>: Defines a clickable button

CSS Properties

CSS properties style the appearance of HTML elements.

  • font-size: Is use for set the font size
  • color: Is use to set the text color
  • background-color: Is use to set the background color
  • margin: Is use to set the margin around an element
  • padding: Is use to set the padding inside an element
  • text-align: Aligns the text content
  • border: Is use to set the border of an element
  • display: Specifies the display behavior of an element

Extra Tags

Additional HTML tags for special purposes.

  • <span>: Defines a generic inline container
  • <meta>: Defines metadata about the HTML document
  • <hr>: Defines a thematic break
  • <em>: Defines emphasized text
  • <strong>: Defines strong importance
  • <link>: Defines a link to an external resource
  • <br>: Defines a line break
  • <title>: Defines the document's title
